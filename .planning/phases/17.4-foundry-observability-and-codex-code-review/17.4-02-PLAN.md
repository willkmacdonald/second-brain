  In `backend/src/second_brain/warmup.py`:

  1. Change the function signature to accept a recreate callback:
     ```python
     async def agent_warmup_loop(
         clients: list[tuple[str, AzureAIAgentClient]],
         interval_seconds: int,
         on_recreate: Callable[[str, AzureAIAgentClient], None] | None = None,
     ) -> None:
     ```
     Add `from collections.abc import Callable` import.

  2. Track consecutive failures per agent:
     ```python
     failure_counts: dict[str, int] = {name: 0 for name, _ in clients}
     MAX_CONSECUTIVE_FAILURES = 3
     ```

  3. In the success path, reset the counter: `failure_counts[name] = 0`

  4. In the failure path, increment and check:
     ```python
     failure_counts[name] = failure_counts.get(name, 0) + 1
     if failure_counts[name] >= MAX_CONSECUTIVE_FAILURES:
         logger.error(
             "Warmup: %s failed %d consecutive pings -- attempting client recreation",
             name, failure_counts[name],
         )
         try:
             new_client = AzureAIAgentClient(
                 credential=client._credential,  # Access from existing client
                 project_endpoint=client._project_endpoint,
                 agent_id=client._agent_id,
                 should_cleanup_agent=False,
                 middleware=client._middleware,
             )
             # Replace in the clients list
             for i, (n, _) in enumerate(clients):
                 if n == name:
                     clients[i] = (name, new_client)
                     break
             if on_recreate:
                 on_recreate(name, new_client)
             failure_counts[name] = 0
             logger.info("Warmup: %s client recreated successfully", name)
         except Exception:
             logger.error("Warmup: failed to recreate %s client", name, exc_info=True)
     ```
