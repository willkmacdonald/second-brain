```python
async def get_routing_context(self) -> str:
```

**3. Add manage_destination tool:**
```python
@tool(approval_mode="never_require")
async def manage_destination(
    self,
    action: Annotated[str, Field(description="Action: 'create', 'rename', 'remove'")],
    name: Annotated[str, Field(description="Destination name (e.g., 'Costco')")],
    slug: Annotated[str, Field(description="URL-safe lowercase slug (e.g., 'costco')")],
    destination_type: Annotated[str | None, Field(description="Type: 'physical' or 'online'. Infer from name if not sure.")] = None,
    new_name: Annotated[str | None, Field(description="New display name (rename only)")] = None,
    new_slug: Annotated[str | None, Field(description="New slug (rename only)")] = None,
) -> str:
    """Create, rename, or remove an errand destination."""
```
- **create**: Check if slug already exists (query by slug). If exists, return "Destination '{name}' already exists." Create DestinationDocument with the provided fields. Default type to "physical" if not provided.
- **rename**: Find destination by slug, update displayName (and optionally slug). If slug changes, do NOT move errand items (that's a future enhancement -- just warn). Return confirmation.
- **remove**: Find destination by slug. Check if errand items exist in that destination partition. If items exist, return "'{name}' has {N} items. Where should they go? (provide a new destination slug)" -- do NOT delete yet. If no items, delete the DestinationDocument and return confirmation.

**4. Add manage_affinity_rule tool:**
```python
@tool(approval_mode="never_require")
async def manage_affinity_rule(
    self,
    action: Annotated[str, Field(description="Action: 'create', 'update', 'delete'")],
    natural_language: Annotated[str, Field(description="Rule in natural language (e.g., 'chicken always goes to Agora')")],
    item_pattern: Annotated[str, Field(description="Primary item/category pattern (e.g., 'chicken')")],
    destination_slug: Annotated[str, Field(description="Target destination slug (e.g., 'agora')")],
    rule_type: Annotated[str, Field(description="Type: 'item', 'category', or 'entity'")],
    exceptions: Annotated[list[dict] | None, Field(description="Exceptions: [{'pattern': 'fish', 'destinationSlug': 'nicks_fishmarket'}]")] = None,
    auto_saved: Annotated[bool, Field(description="True if this rule was auto-saved from a HITL routing answer")] = False,
) -> str:
    """Create, update, or delete an affinity routing rule.

    Returns confirmation on success, or conflict notification if an
    existing rule covers the same item pattern.
    """
```
- **create**: Query existing rules. Check if any rule has the same itemPattern (case-insensitive). If conflict found, return "'{item_pattern}' currently routes to '{existing_dest}'. Say 'yes change it' to update, or 'no keep it' to cancel." (The LLM will handle the follow-up.) If no conflict, create AffinityRuleDocument.
- **update**: Find rule by itemPattern, update destination and other fields. Set updatedAt.
- **delete**: Find rule by itemPattern, delete the document. Return confirmation.

```python
@tool(approval_mode="never_require")

    query_text: Annotated[str, Field(description="What the user asked, e.g., 'where does chicken go?' or 'what are my rules?'")],
) -> str:
    """Answer questions about routing rules and destinations.

    Returns formatted information about rules matching the query.
    For 'what are my rules?' returns all rules. For 'where does X go?'

```
- Load all rules from AffinityRules container
- Load all destinations from Destinations container
- Return formatted text. For "what are my rules?": list all rules grouped by destination. For "where does X go?": return the matching rule or "No rule found for '{X}'. You can create one by saying '{X} goes to [destination]'."
- The LLM will interpret the query and pass it to this tool. The tool returns raw data; the LLM formats the response to the user.

**Important notes:**

- All tools are `async def` and use `await` for Cosmos operations
- Use `Annotated[..., Field(description=...)]` for all tool parameters per existing pattern
- Import DestinationDocument and AffinityRuleDocument from documents.py
