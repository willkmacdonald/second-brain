```python
async def create_container() -> None:
    endpoint = os.environ.get("COSMOS_ENDPOINT")
    credential = DefaultAzureCredential()
    client = CosmosClient(url=endpoint, credential=credential)
    try:
        database = client.get_database_client(DATABASE_NAME)
        try:
            await database.create_container(
                id=CONTAINER_NAME,
                partition_key={"paths": [PARTITION_KEY], "kind": "Hash"},
            )
        except CosmosResourceExistsError:
            logger.info("Container '%s' already exists", CONTAINER_NAME)
    finally:
        await client.close()
        await credential.close()
```

       - `id: str = Field(default_factory=lambda: str(uuid4()))` -- UUID
       - `userId: str = "will"` -- partition key
       - `signalType: str` -- one of: "recategorize", "hitl_bucket", "errand_reroute", "thumbs_up", "thumbs_down"

       - `originalBucket: str` -- what classifier assigned
       - `correctedBucket: str | None = None` -- what user changed it to (None for thumbs_up)

    2. `GoldenDatasetDocument(BaseModel)`:
       - `id: str = Field(default_factory=lambda: str(uuid4()))` -- UUID
       - `userId: str = "will"` -- partition key
       - `inputText: str` -- the capture text to classify
       - `expectedBucket: str` -- known-correct bucket label
       - `source: str` -- one of: "manual", "promoted_feedback", "synthetic"
       - `tags: list[str] = Field(default_factory=list)` -- optional grouping: "edge_case", "voice", "recipe"
       - `createdAt: datetime = Field(default_factory=lambda: datetime.now(UTC))`
       - `updatedAt: datetime = Field(default_factory=lambda: datetime.now(UTC))`
       Add docstring: "Individual test case for classifier evaluation. One document per test case. Partition key: /userId"

    3. `EvalResultsDocument(BaseModel)`:
       - `id: str = Field(default_factory=lambda: str(uuid4()))` -- UUID
       - `userId: str = "will"` -- partition key
       - `evalType: str` -- "classifier" or "admin_agent"
       - `runTimestamp: datetime` -- when the eval started
       - `datasetSize: int` -- how many test cases were evaluated
       - `aggregateScores: dict` -- e.g., {"accuracy": 0.92, "precision": {...}, "recall": {...}}
       - `individualResults: list[dict]` -- e.g., [{"input": "...", "expected": "...", "actual": "...", "correct": True}]

       - `createdAt: datetime = Field(default_factory=lambda: datetime.now(UTC))`
