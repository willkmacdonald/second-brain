## Decisions Made
- Font family strings use exact @expo-google-fonts export names (e.g. `InstrumentSerif_400Regular_Italic`) since React Native uses registered font names, not CSS font stacks
- useTheme is a plain function returning the theme object constant, not a Context provider -- keeps it simple per D-05 since there's no dynamic theming
- SplashScreen.preventAutoHideAsync() placed at module scope after initSentry() to ensure splash stays visible during font loading
- All 3 Stack.Screen header styles updated to design tokens in same commit as font loading
