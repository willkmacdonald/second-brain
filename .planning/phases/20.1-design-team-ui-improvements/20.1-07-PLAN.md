  <what-built>
Full visual reskin of the Second Brain mobile app. All screens (Capture, Inbox, Tasks, Status, Settings) and shared components now use:
- Design tokens from docs/ui-design/screens/tokens.jsx
- Instrument Serif italic headlines for screen titles
- Instrument Sans for body text and labels
- JetBrains Mono for metadata, timestamps, and CapsLabels
- Per-bucket muted colors (People=warm sand, Projects=dusty blue, Ideas=muted violet, Admin=sage)
- Lucide icons for tab bar and UI elements
- Hairline borders (0.5px) instead of 1px
- Dark translucent tab bar with blur effect
  </what-built>
