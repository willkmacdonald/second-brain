    **Step 3: Create `backend/src/second_brain/eval/metrics.py`** with three functions:

    `compute_classifier_metrics(results: list[dict]) -> dict`:
    - Input: list of dicts with keys `predicted`, `expected`, `confidence`, `correct`
    - Computes: `accuracy` (correct/total), per-bucket `precision` and `recall` for buckets {"People", "Projects", "Ideas", "Admin"}
    - For each bucket: TP = predicted==bucket AND expected==bucket; FP = predicted==bucket AND expected!=bucket; FN = predicted!=bucket AND expected==bucket
    - precision[bucket] = TP/(TP+FP) if (TP+FP)>0 else 0.0
    - recall[bucket] = TP/(TP+FN) if (TP+FN)>0 else 0.0
    - Returns `{"accuracy": float, "total": int, "correct": int, "precision": dict, "recall": dict}`
    - Edge case: empty list returns `{"accuracy": 0.0, "total": 0, "correct": 0, "precision": {}, "recall": {}}`

    `compute_confidence_calibration(results: list[dict], bins: int = 5) -> list[dict]`:
    - Bins results by confidence range [0.0-0.2), [0.2-0.4), ..., [0.8-1.0]
    - Last bin includes 1.0 (use `lo <= confidence <= hi` for last bin, `lo <= confidence < hi` for others)
    - Per bin: count, avg_confidence (rounded to 3 decimals), actual_accuracy (rounded to 3 decimals)
    - Returns list of `{"bin": "0.0-0.2", "count": int, "avg_confidence": float, "actual_accuracy": float}`
    - Empty bins are omitted. Empty input returns empty list

    `compute_admin_metrics(results: list[dict]) -> dict`:
    - Input: list of dicts with keys `predicted_destination`, `expected_destination`, `correct`
    - Computes: `routing_accuracy` (correct/total), per-destination accuracy breakdown
    - Returns `{"routing_accuracy": float, "total": int, "correct": int, "per_destination": dict}`
    - Edge case: empty list returns `{"routing_accuracy": 0.0, "total": 0, "correct": 0, "per_destination": {}}`
