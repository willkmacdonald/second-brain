Post-hoc tool detection helper (replaces _count_tool_invocations):
```
def _output_tool_called(response) -> tuple[bool, set[str]]:
    """Inspect response.messages for tool calls.

    Returns (output_tool_fired, all_tools_called).
    Per probe 2: tool calls are in messages with role='tool'.
    The Content blocks inside have a name field for the function called.
    """
    tools_called: set[str] = set()
    for msg in response.messages:
        if getattr(msg, "role", None) == "tool":
            for content in (getattr(msg, "contents", None) or []):
                name = (
                    getattr(content, "name", None)
                    or getattr(content, "function_name", None)
                )
                if name:
                    tools_called.add(name)
    return bool(tools_called & _OUTPUT_TOOL_NAMES), tools_called
```
