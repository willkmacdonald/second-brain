1. **RC -> GA imports + agent calls (F-04):** `client.get_response(stream=True, options=ChatOptions(...))` becomes `agent.run(messages=msg_list, options=ChatOptions(tool_choice='required'), stream=True)`. The streaming iterator yields `AgentResponseUpdate` per probe 1.

2. **Custom span deletion (F-14, D-13):** Delete `with tracer.start_as_current_span("capture_text" / "capture_voice" / "capture_follow_up") as span:` blocks (3 sites).

3. **Safety net deletion + forced_tool_failure SSE sub-code (F-09 + D-03 + D-04):** Delete `_safety_net_file_as_misunderstood` function. Replace each call site with a `forced_tool_failure` SSE event when the agent run fails.

4. **Conversation history threading (F-13, P0-1 OUTCOME, P0-2):** Replace `options['conversation_id'] = foundry_thread_id` and `AgentSession(session_id=resolved_id)` with explicit message-list threading. The capture handler reads the inbox doc's `conversationHistory` via `resolve_inbox_conversation_history()` (24-15 helper). The adapter constructs a `list[Message]` from that history, appends the new user turn, calls `agent.run(messages=msg_list, stream=True)`. After the stream completes, the adapter appends the final assistant turn to the history and persists the updated list back to the inbox doc.

5. **Voice function removal:** Delete `stream_voice_capture` entirely.

KEEP: SSE wire format (CLASSIFIED, LOW_CONFIDENCE, MISUNDERSTOOD, UNRESOLVED, ERROR, COMPLETE event types). KEEP: `_parse_args`, `_parse_result` helpers. KEEP: 60-second timeout via asyncio.timeout. KEEP: rate limiter and per-request error handling.

GA streaming pattern under P0-1 OUTCOME Option A (per probe 1):
```python
from agent_framework import Agent, Message, ChatOptions
from second_brain.cosmos.inbox_conversation_history import (
    ConversationTurn,
    resolve_inbox_conversation_history,
)

# Load conversation history from inbox doc (returns [] for legacy docs with warning)
history = resolve_inbox_conversation_history(inbox_doc)

# Build explicit Message list — historical turns + the new user turn
msg_list = [Message(role=t.role, contents=[t.content]) for t in history]
msg_list.append(Message(role="user", contents=[new_user_text]))

options = ChatOptions(tool_choice="required")  # string form per probe 3

accumulated_assistant_text = ""

try:
    async with asyncio.timeout(60):
        async for update in agent.run(msg_list, options=options, stream=True):
            for content in getattr(update, "contents", None) or []:
                if content.type == "function_call":
                    yield encode_sse({"type": "tool_call", ...})
                elif content.type == "function_result":
                    parsed = _parse_result(content.result)
                    if parsed["status"] == "filed":
                        yield encode_sse({"type": "CLASSIFIED", ...})
                    elif parsed["status"] == "low_confidence":
                        yield encode_sse({"type": "LOW_CONFIDENCE", ...})
                    elif parsed["status"] == "misunderstood":
                        yield encode_sse({"type": "MISUNDERSTOOD", ...})
                    ...
            text_delta = getattr(update, "text", "")
            if text_delta:
                accumulated_assistant_text += text_delta
                yield encode_sse({"type": "text", "content": text_delta})

    # P0-1 OUTCOME Option A: append the new turns to history and persist
    history.append(ConversationTurn(role="user", content=new_user_text))
    if accumulated_assistant_text:
        history.append(ConversationTurn(role="assistant", content=accumulated_assistant_text))
    inbox_doc["conversationHistory"] = [t.model_dump() for t in history]
    await cosmos_manager.upsert_inbox(inbox_doc)

    # COMPLETE event — thread_id is a fresh UUID for mobile backward compat (no server meaning)
    yield encode_sse({
        "type": "COMPLETE",
        "thread_id": str(uuid.uuid4()),
    })

except Exception as exc:
    logger.error("Forced tool failure", exc_info=True, extra=...)
    yield encode_sse({
        "type": "ERROR",
        "sub_code": "forced_tool_failure",
        "message": "Classifier could not file this capture.",
    })
```
