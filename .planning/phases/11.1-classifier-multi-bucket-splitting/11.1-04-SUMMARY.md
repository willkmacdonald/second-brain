## Files Created/Modified

No code files were modified. The change was entirely within the AI Foundry portal Classifier agent instructions.

**Portal change:** Replaced the multi-intent detection section starting with "Before classifying, check if the capture contains MULTIPLE distinct intents" with a 6-rule section starting with "After determining the primary bucket for the capture, check if it also contains a SECOND distinct intent."

Key instruction changes:
- Rule 1: Most captures are single-intent; classify immediately
- Rule 2: Multi-intent requires explicit conjunction markers ("and also", "plus", "oh and")
- Rule 5: When in doubt, classify as single item; never ask for clarification due to conversational phrasing
- Rule 6: Conversational phrasing ("I need to buy dog food") treated identically to imperative ("buy dog food")

## Decisions Made
- Changed analysis order from "before classifying" to "after determining primary bucket" to prevent the classifier from second-guessing simple captures
- Listed explicit conjunction markers as the only trigger for multi-intent splitting, removing ambiguity
- Added Rule 6 with explicit conversational vs imperative equivalence examples
