  <action>
    Create `backend/src/second_brain/api/shopping_lists.py` following the exact same patterns as `inbox.py`:

    **Store display name mapping** (per CONTEXT.md locked decision):
    ```python
    STORE_DISPLAY_NAMES: dict[str, str] = {
        "jewel": "Jewel-Osco",
        "cvs": "CVS",
        "pet_store": "Pet Store",
        "other": "Other",
    }
    ```
    For any store not in this dict, use `store.replace("_", " ").title()` as fallback.

    **Pydantic response models:**
    - `ShoppingItemResponse`: id (str), name (str), store (str)
    - `StoreSection`: store (str), displayName (str), items (list[ShoppingItemResponse]), count (int)
    - `ShoppingListResponse`: stores (list[StoreSection]), totalCount (int)

    **GET /api/shopping-lists** endpoint:
    - Get cosmos_manager from request.app.state (503 if None)
    - Get the "ShoppingLists" container
    - Loop through KNOWN_STORES, query each partition individually: `container.query_items(query="SELECT * FROM c", partition_key=store)`
    - For each store with items, create a StoreSection with the display name from STORE_DISPLAY_NAMES
    - CRITICAL: Sort stores by item count descending (most items first) per CONTEXT.md locked decision
    - Only include stores that have items (no empty sections)
    - Return ShoppingListResponse with totalCount

    **DELETE /api/shopping-lists/items/{item_id}** endpoint:
    - Accept `store` as a required query parameter: `store: str = Query(..., description="Store name (partition key)")`
    - Validate store is in KNOWN_STORES (400 if not)
    - Get cosmos_manager from request.app.state (503 if None)
    - Call `container.delete_item(item=item_id, partition_key=store)`
    - Catch CosmosResourceNotFoundError -> 404
    - Return 204 No Content on success
    - Log the deletion with logger.info

    CRITICAL: The ShoppingLists container uses `/store` as partition key, NOT `/userId`. Every query_items call MUST use `partition_key=store` (the store name string like "jewel"), NOT `partition_key="will"`. Every delete_item call MUST pass `partition_key=store`.

    **Register in main.py:**
    Add import and include_router for shopping_lists_router, following the existing pattern (inbox_router, capture_router, health_router).
